# random-language-compiler
Example of program language for training assistance. Comes with built-in IDE. 

## Example usage

```
данные систолического_давления = 120 125 122 134 150 145 155
данные диастолического_давления = 55 57 53 67 75 70 80
значения(систолического_давления и диастолического_давления)

данные диастолического_давления_2 = 55 57 55 65 57
значения(диастолического_давления_2)

данные пульса = 73 78 75 84 110 105 115
данные температуры = 36.6 36.7 36.6 36.8 36.7 36.5 36.8
значения(пульса и температуры)
график(пульса и температуры и диастолического_давления и систолического_давления)

тренировка бег :
	вес = 72
	систолическое_давление = 120 125 122 134 150 145 155
	диастолическое_давление = 55 57 53 67 75 70 80
	пульс = 73 78 75 84 110 105 115
	температура = 36.6 36.7 36.6 36.8 36.7 36.5 36.8
	широта = 55.664521 55.665434 55.666411 55.667625 55.668736 55.669618 55.669939
	долгота = 37.482886 37.482110 37.481288 37.480252 37.479310 37.478723 37.479828 

анализировать(бег)
```
